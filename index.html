Hello.

Here are some of the proyects that I have developed using Python, C++, Matlab and others.

C++

Sumo robot: Robot Competition “BOGOTRÓN contest 2019”. (Arduino)

This was a robot designed to satisfy the parameters and needs established at an internal university contest. It had the capacity to move in all directions and rotating.


The structure was designed in 3D and the selection of components was made in order to accomplish the main goal of the contest. An arduino controller was used to develop the code and via Bluetooth was connected to an App that helped as a remote controll

Prototype: Hoist for an automotive production line. (Arduino)

This Project involved the design and assembly from scratch, up to the mechanical and electrical components selection.


The microcontroller was programmed with some assigned sequences, representing an assembly line in an automotive fabric. It had buttons to control the process.

Prototype of an encoder connected to a mobile APP. (Arduino)

Prototype designed to send real time data from the encoder to a mobile APP via Bluetooth.


It involved the design of the structure, components selection, laser cutting, and programming for both, the controller and the mobile APP

Synchronization of a car Engine. (MPC5634M microcontroller)

The objective of this proyect was to read the signal of a crankshaft position sensor through a developed program, using a microcontroller and programming to identify points of interest in the functioning of a motor such as the TDC`s and starting sequence parameters to obtain information regarding the behavior of it.

Matlab / Simulink

Control for an inverted pendulum digital twin in MATLAB.

The CAD model was designed and adapted to Simulink from MATLAB. LQR was designed and used for the control


Human Machine Interface (HMI) made with App Designer. Shows real time data from the digital twin.

Python

Web crawlers.

Many web crawler used to pull and store big amounts of data from diverse websites. It involved the use of different libraries, such as: Pandas, BeautifulSoup, Scrapy, Selenium.

Tia Portal (PLC S7-1500)

Creation of a web server connected to a PLC Siemens S7-1500

The server was coded using html and was linked to the PLC CPU.
